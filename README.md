### Запуск в Docker

файл .env  не должен добавляться, так как там обычно содеражтся важные данные
. Можно добавлять только пример [.env.example](.env.example)
- в спецификации прописаны типы возвращаемых ошибок (нет 500)
- создание uuid производится на go (описать преимущества)
